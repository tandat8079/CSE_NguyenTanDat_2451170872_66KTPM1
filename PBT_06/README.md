# 📋 Phiếu Bài Tập 06 - CSS Frameworks

Dự án này là bài tập thực hành môn học phần CSS Frameworks, tập trung vào việc ứng dụng hai framework phổ biến nhất hiện nay là **Bootstrap 5** và **TailwindCSS** để xây dựng giao diện người dùng (UI) một cách nhanh chóng và chuẩn Responsive.

---

##  Công nghệ sử dụng

* **HTML5:** Xây dựng cấu trúc trang web.
* **Bootstrap 5 (Track A):** Sử dụng Grid system, Utilities và Components (thông qua CDN).
* **TailwindCSS (Track B):** Sử dụng Utility-first classes để style trực tiếp trên HTML (thông qua CDN).

---

##  Cấu trúc thư mục và File

Dự án bao gồm các file chính sau:

###  Lý thuyết & Phân tích
* `answers.md`: Chứa câu trả lời chi tiết cho **Phần A (Đọc hiểu)** và **Phần C (Phân tích)** của cả 2 Track Bootstrap và TailwindCSS.

### 🅱 Track A: Bootstrap 5
* `bootstrap_landing.html`: Trang Landing Page E-commerce sử dụng 100% Bootstrap classes (Navbar, Hero Carousel, Product Grid, Modal,...).
* `bootstrap_dashboard.html`: Trang Admin Dashboard layout với Sidebar cố định, Topbar, Stat cards, Data table và Accordion FAQ.

###  Track B: TailwindCSS
* `tailwind_landing.html`: Trang Landing Page được xây dựng bằng Tailwind utility classes, có hỗ trợ Responsive và Dark Mode.
* `tailwind_components.html`: Bộ sưu tập các UI Components phổ biến (Buttons, Forms, Alerts) với các trạng thái hover/focus.
